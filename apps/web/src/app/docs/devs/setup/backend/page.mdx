# Backend Setup

This section will guide you through configuring the backend service for
development.

## Required Environment Variables

- `DATABASE_URL{:bash}`: Connection string for your PostgreSQL database
- `JWT_SECRET{:bash}`: A secret key used for signing
  [JSON Web Tokens](https://jwt.io/) (JWT).
- `SHARED_JWT_SECTRET{:bash}`: This is a secret that should be set on both the web client and backend. Its used for encrypted communication between the two.

## Run Migrations

You then need to run migrations to set up your database schema.

```bash
pnpm prisma migrate dev
```
